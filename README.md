# avito-multimodal-prediction

## Задача
- Предсказать вес и габариты товара по данным

## Моё решение
1. **Провожу EDA**, строю графики и смотрю на данные (из интересного в height и width часто бывают кргулые и кратные 5 значения)
2. **Настраиваю окружение, определяю конфигурацию и модели**
  - `tf_efficientnet_b4_ns` - для image embedding
  - `cointegrated/rubert-tiny2` - для text embedding
3. **Извлечение признаков**
    - Регулярки
    - Эмбеддинги
    - Параллельно смотрю где у меня большие ошибки и подстраиваю `extract_features_smart` под новые возникающие ошибки
4. **Обучение CatBoost**
    - Обучаю 2 модели с разными параметрами
    - Первую на всех данных что есть
    - Вторую без картинов (чтобы фокус был на text embedding)
5. **Блендинг**
    - По валидации смотрю что где полезно и задаю веса
    - Делаю блендинг и финальный сабмит

## По результатам
Log-MAE = ~0.3201 на тесте

## Что можно исправить и добавить
- Попробовать другие модели для эмбеддингов
- Полностью отказаться от текстовых эмбеддингов, так как они почти не дают прироста
- 2 модели (как моя первая модель) с разными гиперпараметрами и сидами обучить и заблендить (этот вариант я попробовала, он лучше)
